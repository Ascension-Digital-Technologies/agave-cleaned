# Support

This repository is an unofficial cleaned and restructured presentation of Anza Agave. It is not an official Agave support channel and should not be used as the source of truth for validator operation or network incidents.

## Where to go

| Need | Recommended place |
| --- | --- |
| Official Agave source/release guidance | <https://github.com/anza-xyz/agave> |
| Agave / Solana documentation | <https://docs.anza.xyz/> |
| Public cluster information | <https://docs.anza.xyz/clusters> |
| Security vulnerability in Agave | Follow [`SECURITY.md`](SECURITY.md) and upstream Anza guidance |
| Issue with this cleaned layout, docs, or scripts | Open an issue in this cleaned repository |

## Good issues for this repository

- broken links,
- incorrect path mappings,
- unclear documentation,
- script portability bugs,
- repository packaging problems,
- source-integrity tooling issues.

## Not good issues for this repository

- validator outage reports,
- mainnet operational support,
- protocol security disclosures,
- official release requests,
- upstream Agave consensus/runtime behavior questions.

For those, use the official Anza Agave channels and documentation.
